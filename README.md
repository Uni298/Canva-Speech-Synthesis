# Speech Synthesis

## 利用規約

### 1. 本サービスの概要

本アプリは、ユーザーが入力したテキストを、音声合成サービス「ゆくも！」および「VOICEVOX」を利用して音声データに変換し、Canva上のデザインに追加する機能を提供します。

### 2. 知的財産権と利用制限

本アプリを通じて生成される音声の著作権および利用条件は、各音声合成エンジンおよびキャラクターの提供元に帰属します。ユーザーは、以下の各規約を個別に遵守する必要があります。

* **「ゆくも！」を利用する場合**
* [ゆくも！利用規約](https://www.yukumo.net/)（AquesTalkライセンス）に従ってください。
* **注意：** 個人の非営利目的を除き、商用利用（収益化されたYouTube動画等を含む）には別途ライセンスの購入が必要な場合があります。


* **「VOICEVOX」を利用する場合**
* [VOICEVOX利用規約](https://voicevox.hiroshiba.jp/term/)および各登場キャラクター（雀松朱音等）の個別規約に従ってください。
* 利用の際は、指定されたクレジット表記（例：「VOICEVOX:雀松朱音」）を行う必要があります。



### 3. 禁止事項

ユーザーは、以下の行為を行ってはなりません。

* 法令または公序良俗に反する内容の音声を生成すること。
* 各音声合成サービスの規約で禁止されている表現（特定の個人・団体への誹謗中傷、政治・宗教活動等）に利用すること。
* 本アプリの解析、改ざん、または過度な負荷をかける行為。

### 4. 免責事項

当方は、本アプリの利用により生じた損害、および生成された音声の内容に関する第三者とのトラブルについて、一切の責任を負いません。各サービスの仕様変更により、予告なく機能が停止する場合があります。

## プライバシーポリシー

### 1. 取得する情報と利用目的

本アプリは、サービスの提供にあたり以下の情報を取得・利用します。

* **テキストデータ**: 音声合成処理のため。入力されたテキストは外部API（ゆくも！およびVOICEVOX関連サーバー）へ送信されます。
* **Canvaユーザー識別子**: アプリの動作制御および利用状況の統計把握のため。
* **利用ログ**: エラーの診断およびサービス向上のため。

### 2. データの管理と外部送信

* **外部送信**: 本アプリは、音声生成のために「ゆくも！」および「VOICEVOX」のAPIサーバーを利用します。送信されたデータは音声生成の目的のみに使用されます。
* **保存期間**: 生成された音声データおよび入力テキストは、処理完了後速やかにサーバーから削除され、当方が永続的に保存することはありません。

### 3. 第三者提供

当方は、法令に基づく場合を除き、ユーザーの同意なく個人情報を第三者に提供することはありません。
